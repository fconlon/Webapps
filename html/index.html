Titanic Disaster Visualization

This app uses the Aframe javascript library to provide a Virtual Reality visualization of the survivor data from the titanic disaster. Clicking on any cylinder will apply a filter to the data and adjust the heights of all of the cylinders in the visualization. For example, clicking on the cylinder labeled "Died" will filter the data to show only the passengers that died in the disaster. Clicking the cylinder a second time will remove the filter from the data.
